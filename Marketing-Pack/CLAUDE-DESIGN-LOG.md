## How to use these in production

### LinkedIn paid ads
1. Take a screenshot of the rendered HTML at exactly 1200×627 (use device toolbar in Chrome dev tools to set viewport precisely)
2. Save as PNG
3. Upload to LinkedIn Campaign Manager
4. Pair with the body copy from `AD-COPY-LIBRARY.md` Variant A/B/C

### Twitter/X promoted
1. Screenshot at 1600×900
2. Upload to X Ads Manager
3. Pair with body copy from `TWITTER-X-POSTS.md` themes 4-6 (brand/company voice)

### Instagram feed
1. Screenshot at 1080×1080
2. Upload via Buffer / native scheduler
3. Caption from `INSTAGRAM-CAPTIONS.md` static post variants

### Email signature
1. Export the email banner as PNG
2. Embed in email signature (Outlook, Gmail, Apple Mail)
3. Set max-width 600px

## Iteration plan

After 2 weeks of running these creatives:
1. Review CTR by ad
2. Kill bottom 50% by performance
3. Generate 3-5 new variants of the top performer (different headline, different visual emphasis)
4. Run another A/B for 2 weeks
5. Repeat

Best practice: rotate creative every 4 weeks to fight banner blindness.
