## A/B testing protocol

For each ad set:
1. Run 3-5 variants for 7 days at $50/day
2. Kill the bottom 50% by CTR
3. Iterate the top performers — change one variable per test (headline, image, CTA, audience)
4. Scale winners that hit >2% CTR + <$80 CPL
5. Sunset variants that drop below 1% CTR for 3 days

## What to track

| Metric | Target |
|---|---|
| CTR | >2% (LinkedIn), >1% (Meta) |
| Cost per click | <$15 (LinkedIn), <$8 (Meta), <$5 (Search) |
| Cost per lead | <$80 |
| Lead → call | >25% |
| Call → close | >15% |
| LTV : CAC | >3:1 |
